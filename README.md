# 面经网站 (Interview Platform)

这是一个专注于分享和总结面试经验的面经网站。本项目采用前后端分离架构进行开发。

## 项目架构
详细的目录结构和技术栈设计，请参阅： 
[docs/project_structure.md](docs/project_structure.md)

## 快速管理

我们提供了快捷的 Bash 脚本以便于在服务端快速启停开发/生产服务（具体指令请参阅脚本内部注释或根据选用的框架自行配置）：

- 启动项目：
  ```bash
  ./start.sh
  ```
- 停止项目：
  ```bash
  ./stop.sh
  ```
